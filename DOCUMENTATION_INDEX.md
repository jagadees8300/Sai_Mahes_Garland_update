## 🚀 Getting Started

### 1. First Time Setup
```bash
cd frontend
npm install
npm start
```
See [QUICKSTART.md](./QUICKSTART.md) for details

### 2. Update Contact Information
Edit these files:
- `src/components/ContactModal.jsx`
- `src/components/SocialLinks.jsx`

See [SOCIAL_MEDIA_CONFIG.md](./SOCIAL_MEDIA_CONFIG.md) for help

### 3. Test Everything
- Open http://localhost:3000
- Test all buttons
- Check animations
- Test on mobile (browser dev tools)

### 4. Deploy
```bash
npm run build
# Deploy the build folder
```
